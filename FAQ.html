Frequently Asked Questions

Q: What is this project about?

This is a decompilation project for TwinBee PARADISE in Donburi Island and TwinBee's Exciting Wonderland. The overall goal is to extract the original source code from the complied code in Macromedia Director and the binaries. When the original source code is extracted from the game, it's going to be reversed enginnered so that people can have a better understanding of how the game work overall.

Q: Why are you decompiling this game?

The reason why I decided to decompile this game is because the TwinBee series never really got noticed outside of Japan and doing something like this coud reach more fans or new fans. Working with Macromedia Director based games is especially challenging for me since the scripting language "Lingo" is no longer supported and is notoriously difficult to reverse engineer. Macromedia Director is an old software and there's many better game engines that can make the game run without any runtime issues.

By decompiling this game, we can understand how the game works based on Lingo script and behaviors. It will make it eaiser to reverse engineer the Lingo script into a different programming langauge that is widely used.

As for TwinBee's Exciting Wonderland, there's a lot of widgets and screensavers that have to be extracted since they aren't has widely used now, but there is a lot of extra content that does have to be saved or experienced.

Q: How can I contribute to the project?

You can:

  • Help with testing or debugging the game
  • Translate any Japanese scripts or text
  • Join the Discord and share the project with others!

Q: Do I need coding experience to help?

Not necessarily. Even if you can’t reverse engineer raw code, you are always welcome to help with documentation, research, and organization. HTML isn't a OOP langauge and it's a fairly easy langauge to understand (It's like making a word document).

If you are experienced in C or used Ghidra, you are welcome to help with decompiling TwinBee's Exciting Wonderland.

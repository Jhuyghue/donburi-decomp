TwinBee PARADISE in Donburi Island Decompilation

A full game decompilation of TwinBee PARADISE in Donburi Island &
TwinBee's Exciting Wonderland for Windows 95 on PC.

This project aims to document the entire game logic and files of both discs of TwinBee PARADISE in Donburi Island and TwinBee's Exciting Wonderland. The ultimate objective is to reconstruct the source code in a readable form. TwinBee PARADISE in Donburi Island will consist of extracting Lingo code in Macromedia Director. TwinBee's Exciting Wonderland will consist of extracting the binaries from the files and matching them. If the full decompilations are successful, there will be a attempt to recompile and port the games on modern operating systems like Windows 10/11.

If you want to learn a little more about this project or have any questions, feel free to visit the frequently asked questions page to get them answered.

This project is currently a work in progress. The documentation is going to be constantly evolving and the code is going to be edited overtime.
